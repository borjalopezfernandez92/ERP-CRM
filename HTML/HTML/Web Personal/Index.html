Página personal de práctica de HTML, CSS y JavaScript.

Recibe la bienvenida a ésta página de práctica donde intento sacarle el mayor partido posible a la teoría que estudiamos sobre HTML, CSS y próximamente JavaScript.

Se trata de una página web con 4 apartados: Home, Biografía, Portfolio y Contacto. En la página Home se da la bienvenida al usuario. En la Biografía se lista una pequeña descripción del creador de ésta página. En el portfolio podremos acceder a trabajos realizados anteriormente y en el contacto podremos escribir un mensaje al creador de la página web.
